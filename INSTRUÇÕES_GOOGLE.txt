============================================================
  CONFIGURAR GOOGLE SHEETS NO BOT
============================================================

PASSO 1: Criar projeto no Google Cloud
---------------------------------------
1. Acesse: https://console.cloud.google.com
2. Crie um novo projeto (ex: "sobrinde-bot")
3. No menu, vá em "APIs e serviços" → "Biblioteca"
4. Ative a "Google Sheets API"
5. Ative também a "Google Drive API"

PASSO 2: Criar Service Account
--------------------------------
1. Vá em "APIs e serviços" → "Credenciais"
2. Clique em "Criar credenciais" → "Conta de serviço"
3. Dê um nome (ex: "sobrinde-sheets")
4. Clique em "Concluído"
5. Clique na conta recém-criada
6. Vá na aba "Chaves" → "Adicionar chave" → "Criar nova chave"
7. Escolha formato JSON → Baixe o arquivo

PASSO 3: Compartilhar a planilha com a Service Account
-------------------------------------------------------
1. Abra o arquivo JSON baixado e copie o campo "client_email"
   (algo como: sobrinde-sheets@sobrinde-bot.iam.gserviceaccount.com)
2. Abra sua planilha do Google Sheets
3. Clique em "Compartilhar"
4. Cole o client_email e dê permissão de EDITOR
5. Clique em "Enviar"

PASSO 4: Adicionar as variáveis no Render
------------------------------------------
1. Acesse: https://dashboard.render.com/worker/srv-d8bcc0f7f7vs73bv5mhg
2. Vá em "Environment" → "Edit"
3. Adicione as variáveis:

   SPREADSHEET_ID = 1LoFAGupt_uB0fPT2MDQbRn4_NFvciWH8VwOIXyFNG9Y

   GOOGLE_CREDENTIALS_JSON = (conteúdo do arquivo JSON em UMA LINHA)
   → Para converter para uma linha: abra o JSON no Notepad,
     selecione tudo e use Ctrl+H para substituir quebras de linha

4. Salve e faça novo deploy

============================================================
  COMO USAR O BOT PARA ATUALIZAR O BI
============================================================

1. Exporte os relatórios PDF do Demander
   (Vendas X Mês, Produto, Cliente, Cidade, Estado — para cada vendedor)

2. No Telegram, envie /relatorios ao bot

3. Envie os PDFs um a um (pode enviar vários seguidos)

4. Quando terminar, envie /processar

5. O bot vai extrair todos os dados e atualizar automaticamente:
   • VENDAS X CIDADES
   • VENDAS X ESTADOS
   • VENDAS X MÊS
   • VENDAS X PRODUTOS
   • PRODUTOS CONSOLIDADOS
   • METAS X VENDAS (seção REALIZADO)
